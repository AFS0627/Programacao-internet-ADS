História da Internet e da World Wide Web

A Internet surgiu da necessidade de conectar computadores para trocar informações. Com o passar do tempo surgiu a World Wide Web (WWW), criada por Tim Berners-Lee, permitindo acessar documentos usando navegadores.

Linha do Tempo da Evolução da Internet e da Web

  1. Década de 1960 – Primeiras pesquisas sobre redes de computadores.
  2. 1969 – Criação da ARPANET, considerada a primeira versão da Internet.
  3. 1983 – Criação do protocolo TCP/IP, permitindo comunicação entre redes.
  4. 1983 – Surgimento do DNS, usando nomes ao invés de IP.
  5. 1989 – Tim Berners-Lee propõe a World Wide Web.
  6. 1990 – Desenvolvimento do HTML, HTTP e do primeiro servidor web.
  7. 1991 – Primeiro site da história é publicado.
  8. 1993 – O CERN libera a Web gratuitamente.
  9. 1994 – Fundação do W3C.
  10. 2000 em diante – Redes sociais, streaming e computação em nuvem.

Tim Berners-Lee e o CERN

Em 1989, Tim Berners-Lee trabalhava no CERN e percebeu que era difícil compartilhar informações entre pesquisadores. Então criou um sistema baseado em hipertexto que deu origem à Web.

"A Web foi criada para ser um espaço universal de compartilhamento de informações."
Referência ao trabalho de Tim Berners-Lee.

Conceitos Fundamentais

ARPANET
Primeira grande rede de computadores.
TCP/IP
Conjunto de protocolos usado na Internet.
DNS
Transforma nomes de sites em endereços IP.
HTML
Linguagem usada para criar páginas.
HTTP
Protocolo usado para acessar páginas.
Servidor Web
Computador que armazena sites.
Navegador
Programa usado para acessar páginas.

Web 1.0 x Aplicações Web Modernas

Web 1.0

A Web 1.0 era formada por páginas estáticas. O usuário apenas lia o conteúdo.

Aplicações Web Modernas

Hoje as páginas são dinâmicas e permitem comentários, redes sociais, streaming, compras e muito mais.

O HTML5 trouxe novos recursos para as páginas. Atualmente bilhões de dispositivos podem se comunicar usando a Internet.

O protocolo TCP/IP possui diferentes camadas, sendo a camada IP responsável pelo endereçamento dos dispositivos.
