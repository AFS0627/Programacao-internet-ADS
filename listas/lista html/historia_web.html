História da Internet e da World Wide Web

A Internet surgiu a partir da necessidade de conectar computadores para troca de informações. Com o passar dos anos, novas tecnologias foram desenvolvidas até o surgimento da World Wide Web (WWW), criada por Tim Berners-Lee, permitindo o acesso a documentos por meio de navegadores.

Linha do Tempo da Evolução da Internet e da Web

  1. Década de 1960 – Surgimento das primeiras pesquisas sobre redes de computadores.
  2. 1969 – Criação da ARPANET, considerada a precursora da Internet.
  3. 1983 – Adoção do protocolo TCP/IP, tornando possível a comunicação padronizada entre redes.
  4. 1983 – Criação do DNS (Domain Name System), permitindo utilizar nomes de domínio em vez de endereços IP.
  5. 1989 – Tim Berners-Lee, pesquisador do CERN, propõe a criação da World Wide Web.
  6. 1990 – Desenvolvimento do primeiro servidor web, do protocolo HTTP e da linguagem HTML.
  7. 1991 – Publicação do primeiro site da história e disponibilização da WWW ao público.
  8. 1993 – O CERN libera a tecnologia da Web para uso gratuito.
  9. 1994 – Fundação do W3C (World Wide Web Consortium), responsável pela padronização da Web.
  10. 2000 em diante – Surgimento das aplicações Web modernas, redes sociais, streaming e computação em nuvem.

Tim Berners-Lee e o CERN

Em 1989, Tim Berners-Lee, trabalhando no CERN, observou dificuldades no compartilhamento de informações entre pesquisadores. Sua solução foi criar um sistema baseado em hipertexto, que deu origem à World Wide Web.

"A Web foi criada para ser um espaço universal de compartilhamento de informações."
Referência ao trabalho desenvolvido por Tim Berners-Lee no CERN.

Conceitos Fundamentais

ARPANET
Primeira rede de computadores de grande escala, criada pelo Departamento de Defesa dos Estados Unidos.
TCP/IP
Conjunto de protocolos responsável pela comunicação entre dispositivos conectados à Internet.
DNS
Sistema que converte nomes de domínio em endereços IP.
HTML
Linguagem de marcação utilizada para estruturar páginas Web.
HTTP
Protocolo utilizado para transferência de páginas e recursos da Web.
Servidor Web
Computador responsável por armazenar e disponibilizar páginas para os usuários.
Navegador
Programa utilizado para acessar e visualizar páginas da Web.

Web 1.0 x Aplicações Web Modernas

Web 1.0

A Web 1.0 era formada principalmente por páginas estáticas. Os usuários apenas consumiam conteúdo, com pouca ou nenhuma interação.

Aplicações Web Modernas

As aplicações atuais são dinâmicas e interativas, permitindo comentários, redes sociais, streaming, compras online e colaboração em tempo real.

Exemplo de evolução: HTML5 trouxe novos recursos multimídia, enquanto protocolos modernos utilizam versões atualizadas do HTTP. Redes atuais podem conectar bilhões de dispositivos e armazenar grandes volumes de dados utilizando tecnologias de nuvem e bancos de dados.

O protocolo TCP/IP é composto por diferentes camadas, incluindo a camada IP, responsável pelo endereçamento dos dispositivos na rede.
