Protocolo HTTP

O HTTP (HyperText Transfer Protocol) é o protocolo utilizado para a comunicação entre navegadores e servidores na Web. Sempre que um usuário acessa um site, uma troca de mensagens HTTP ocorre para solicitar e receber informações.

O papel do HTTP na Web

Quando digitamos o endereço de um site no navegador, o navegador envia uma requisição HTTP para um servidor web. O servidor processa o pedido e retorna uma resposta HTTP contendo os dados solicitados, como páginas HTML, imagens, vídeos ou arquivos.

Navegador → Requisição HTTP → Servidor

Servidor → Resposta HTTP → Navegador

Cliente, Servidor, Requisição e Resposta

Cliente
Aplicação que solicita recursos ao servidor, normalmente um navegador web.
Servidor
Computador responsável por armazenar e disponibilizar páginas e serviços.
Requisição HTTP
Mensagem enviada pelo cliente solicitando algum recurso.
Resposta HTTP
Mensagem enviada pelo servidor contendo o resultado da solicitação.

Métodos HTTP

GET

O método GET é utilizado para solicitar informações ao servidor. É o método mais comum ao acessar páginas da Web.

Exemplo: consultar uma página de notícias ou visualizar um produto em uma loja virtual.

POST

O método POST é utilizado para enviar dados ao servidor. É muito usado em formulários de cadastro, login e envio de informações.

Exemplo: enviar um formulário de contato ou criar uma nova conta em um site.

Exemplo de Requisição HTTP

 GET /index.html HTTP/1.1 Host: www.exemplo.com User-Agent: Mozilla/5.0 Accept: text/html

Nesse exemplo, o navegador solicita ao servidor o arquivo index.html.

Exemplo de Resposta HTTP

 HTTP/1.1 200 OK Content-Type: text/html <html> <body> Página encontrada. </body> </html>

O código 200 OK indica que a requisição foi processada com sucesso.

Códigos de Status HTTP

Principais Códigos HTTP
Código Nome Significado
200 OK A requisição foi processada com sucesso.
301 Moved Permanently O recurso foi movido permanentemente para outro endereço.
302 Found Redirecionamento temporário para outro endereço.
400 Bad Request A requisição enviada pelo cliente possui erro.
403 Forbidden O acesso ao recurso foi negado pelo servidor.
404 Not Found O recurso solicitado não foi encontrado.
500 Internal Server Error O servidor encontrou um erro interno.
Códigos de status utilizados na comunicação HTTP.

Resumo

O HTTP é um dos protocolos fundamentais da Internet moderna. Ele permite que navegadores e servidores troquem informações de forma padronizada, tornando possível o funcionamento de sites, sistemas online, redes sociais e aplicações web.
