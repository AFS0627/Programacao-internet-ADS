css

linguagem de marcação para estilização e definição de comportamento sobre o html

pode ser usado de 3 maneiras diferentes

externo
css externo é definido utilizando a tag link no inicio do documento html na seção de head alt link rel="stylesheet" href="style.css" >
css inline

model box (modelo em caixa)

o html e css possuem um conceito central que é o modelo em caixa, um elemento é definido dentro do outro tendo uma estrutura de container
