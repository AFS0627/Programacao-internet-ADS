Display do elementos html

Os elementos html possuem um exibição ou display inline ou block

block - é uma exibição que inicia na margem esquerda até a direita, em que podemos determinar altura e largura do elemento.Por padrão, no html, elementos com exibição block não dividem espaço na horizontal com outros elementos

inline - é uma exibição que permite que o elemento seja apresentado dentro de uma linha com outros elementos como um elemento textual. exemplo

  1. item ordenado
  2. item ordenado
  3. item ordenado
  • item 1 não ordenado
  • item 2 não ordenado
  • item 3 não ordenado
